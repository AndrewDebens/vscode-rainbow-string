## Rainbow String 

Add rainbow effect to quoted string

![](https://raw.githubusercontent.com/wk-j/vscode-rainbow-string/master/images/Rainbow.png)

## Configurations

```json
"rainbowString": {
    "extensions": [
        ".cs",
        ".csx",
        ".fs",
        ".fsx",
        ".js",
        ".fsx",
        ".ts",
        ".tsx",
        ".csproj",
        ".fsproj",
        ".cake"
    ]
}
```
